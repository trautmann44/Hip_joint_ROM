# Gymnastics-Based Intervention and Hip Joint Range of Motion in Youth Football Players

## 📌 Project Overview
This project investigates whether gymnastics-based flexibility training can improve hip joint range of motion (ROM) in young football players.

## ❓ Research Question
Can structured gymnastics-inspired exercises improve hip mobility (flexion, extension, abduction) in youth athletes?

## 📊 Data
- Sample: 29 youth football players (mean age = 9.9 ± 1.63 years)
- Design: 3-month intervention study
- Training protocol: 3 × 30-minute flexibility sessions per week

## 🧪 Methods
- Video-based frame analysis for ROM assessment
- Hip joint movements analyzed:
  - Flexion
  - Extension
  - Abduction
- Statistical analysis:
  - Linear mixed-effects models
  - Backward stepwise selection
- Effect sizes reported using Cohen’s d

## 📈 Key Results
The intervention led to statistically significant improvements in all measured movement directions:

- Flexion: significant increase (p < 0.001, d = 0.98–1.37) = 10.0 % improvement
- Extension: significant increase (p < 0.001, d = 0.98–1.37) = 12.8 % improvement
- Abduction: significant increase (p < 0.001, d = 0.98–1.37) = 15.4 % improvement

## 🧠 Interpretation
Gymnastics-based flexibility training appears to be an effective method for improving hip joint mobility in youth football players.

This suggests that movement patterns from gymnastics may be transferable to other sports and useful for injury prevention and performance development.

## 🧰 Tools
R, linear mixed-effects models, video-based motion analysis
